Game Instructions
  1. There are 60 seconds per game
  2. A name of the countries pops up randomly so you type it in the input box. Note letter case and space matter
  3. You get 10 points per correct typing
  4. At the end of each game, you receive a medal depending on the score
    • Gold Medal - Over 230 points
    • Silver Medal - Between 180 and 220 points
    • Bronze Medal - Below 170 points
  6. To play the game, click Play button below input box. Input box will be focused automatically so you don't need to click it

*** IMPORTANT ***
Speed Typing game is available on Laptop / Desktop ONLY as a key board is required to play it
