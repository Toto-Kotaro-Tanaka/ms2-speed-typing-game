Game Instructions
  • 60 seconds per game
  • Country names pop up randomly & type them in input box. Note letter case and space matter
  • 10 points per correct typing
  • You get a medal at the end of each game;
    1. Gold Medal - Over 230 points
    2. Silver Medal - Between 180 and 220 points
    3. Bronze Medal - Below 170 points
  • To play game, click Play button below input box. Input box will be focused automatically so no need to click it
